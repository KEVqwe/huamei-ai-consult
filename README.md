# 上海华美 AI 咨询智能体 Demo

面向 C 端求美者的 AI 售前咨询智能体，目标是替代人工售前咨询：接待求美者 → 了解需求 → 依据院内价目/政策推荐项目 → 引导预约到院面诊。

## 目录结构

```
AI医美咨询Demo/
├── 素材/                    # 原始素材（由5个zip解压，全部为图片/视频）
├── knowledge/               # ★ 结构化知识库（由价目表图片视觉识别转录为Markdown）
│   ├── 皮肤美容科价目表.md   # 水光/抗衰/激光/光子/热玛吉/超声炮/脱毛/纹绣/病症等
│   ├── 皮肤美容套餐.md       # 祛斑/抗衰/祛痘/腰腹塑形套餐
│   ├── 微整形科价目表.md     # 玻尿酸/肉毒素(4品牌)/线雕(PPDO·悦升·童颜·鱼骨)/年卡
│   ├── 整形外科价目表.md     # 眼/鼻/胸/吸脂/脂肪填充/改脸型/口唇/耳部
│   ├── 私密项目价目表.md     # Laqueen私密中心全线（手术/注射/光电/套餐）
│   ├── 当月政策与活动.md     # 高定大师季：新客礼/充值礼/限量卡项/老带新/明星产品
│   └── 荟员升级权益.md       # V1雏菊粉卡~V5铃兰黑卡会员体系
├── app/
│   ├── server.js            # 零依赖Node服务：静态页 + /api/chat
│   ├── prompts/system.md    # 咨询顾问人设与合规约束（可直接编辑调优）
│   └── public/index.html    # C端移动风格聊天界面
└── .claude/launch.json      # 一键启动配置
```

## 运行

```bash
node app/server.js          # 默认 http://localhost:3080
```

### 三种模式（按优先级自动切换：DeepSeek > Claude > 本地）

| 模式 | 条件 | 说明 |
|---|---|---|
| **DeepSeek API 模式** | 设置 `DEEPSEEK_API_KEY` | OpenAI兼容接口；模型用 `DEEPSEEK_MODEL` 指定（默认 deepseek-chat），网关地址可用 `DEEPSEEK_BASE_URL` 覆盖 |
| **Claude API 模式** | 设置 `ANTHROPIC_API_KEY` | 模型可用 `CLAUDE_MODEL` 覆盖，默认 claude-sonnet-5 |
| **本地演示模式** | 无任何 API Key | 意图识别（25+医美意图）→ 知识库分节检索 → 规则化咨询应答，可离线演示 |

```powershell
# Windows 启用 DeepSeek 模式
$env:DEEPSEEK_API_KEY = "sk-..."          # DeepSeek开放平台 https://platform.deepseek.com 创建
$env:DEEPSEEK_MODEL   = "deepseek-chat"   # 换成你开通的模型ID（如 v4 pro 对应的ID）
node app/server.js

# 或启用 Claude 模式
$env:ANTHROPIC_API_KEY = "sk-ant-..."
node app/server.js
```

## 智能体能力设计

1. **项目咨询与报价**：所有价格严格来自 knowledge/ 知识库（价目表图片逐张视觉识别转录），不编造。
2. **需求引导**：模糊需求先追问（部位/困扰/预算），再给 1-3 个匹配方案。
3. **营销联动**：自动关联新客礼、本月活动、买二送一、老带新、储值/荟员权益。
4. **留资转化**：识别预约/面诊/地址等高意向信号，引导留下称呼+电话。
5. **合规红线**：不诊断不承诺疗效；手术提示风险；孕哺/未成年建议暂缓；"价格以到院面诊为准"。

## 后续路线建议（Demo → 生产）

- **渠道接入**：企业微信/微信公众号/小程序客服接口（当前 /api/chat 为标准 JSON 接口，易对接）。
- **RAG 升级**：知识库条目化+向量检索（价目表已结构化，可直接建索引），替代整库注入以降低 token 成本。
- **留资闭环**：留资写入 CRM/企微，人工顾问接力；增加转人工按钮。
- **知识库运维**：每月政策更新只需替换 knowledge/*.md（建议由运营用表格维护后导出）。
- **案例图与视频**：素材中的案例图（奇迹水案例库等）和宣传视频可作为对话中的富媒体卡片下发。
- **质检与评测**：抽样对话质检（报价准确率/合规话术/留资率）作为上线门槛。
