# 上海华美 AI 咨询智能体（v1.13）

面向 C 端求美者的 AI 售前咨询智能体，目标是替代人工售前咨询：接待求美者 → 了解需求 → 依据院内价目/政策推荐项目/医生 → 引导预约到院面诊。已部署于 Zeabur（GitHub push 自动发布）。

## 运行

```bash
node app/server.js          # 默认 http://localhost:3080，端口被占自动+1
```

### 三种模式（按优先级自动切换：DeepSeek > Claude > 本地）

| 模式 | 条件 | 说明 |
|---|---|---|
| **DeepSeek API 模式** | 设置 `DEEPSEEK_API_KEY` | OpenAI兼容接口；模型用 `DEEPSEEK_MODEL` 指定，网关可用 `DEEPSEEK_BASE_URL` 覆盖 |
| **Claude API 模式** | 设置 `ANTHROPIC_API_KEY` | 模型可用 `CLAUDE_MODEL` 覆盖，默认 claude-sonnet-5 |
| **本地演示模式** | 无任何 API Key | 30+医美意图识别 → 知识库分节检索 → 短句分段应答，可离线演示 |

公网部署建议设置 `ACCESS_CODE`（访问口令，防接口盗刷）。

## 核心机制

1. **RAG 检索注入**：按最近3条用户消息匹配知识库文档关键词，只注入最相关的4个文档（含目录兜底），省 token 且多轮追问不丢上下文。
2. **真人化对话**：`|||` 分段短消息（每条≤20字）、按问题复杂度的思考延迟、约7字/秒打字节奏、发图前"找图"停顿、开场接入动画；system.md 内含对话样例与表情/追问约束。
3. **模型自主发图**：60张配图目录注入提示词，模型用 `[[图:标签]]` 插图（服务端解析并容错单括号/中文冒号），本地模式走关键词匹配。
4. **报价合规**：价格仅出自知识库；不诊断不承诺疗效；孕哺/未成年劝缓；"以到院面诊为准"。
5. **图片链路**：assets（Q85压缩、原分辨率）→ Zeabur CDN 30天缓存 + ETag；本地开发自动回退素材原图。
